# 🌍 Geoworld

Ever heard of the popular game GeoGuessr?

It’s a super fun way to explore the world and test your geography skills. Geoworld is a free alternative that brings that same experience while giving players more freedom, customization, and multiplayer interaction.

Geoworld is a multiplayer geography game where you can explore real-world locations, guess where you are, and challenge friends or strangers in real time.

---

## ✨ Features

- 🗺️ **Custom Maps**
  - Create your own maps with custom locations
  - Share them publicly with the community
  - Explore maps made by other players

- 🎮 **Multiplayer Gameplay**
  - 1v1 matches against friends
  - Real-time gameplay powered by sockets
  - Competitive scoring system

- 📅 **Daily Challenge**
  - A new challenge every day
  - Compare your score with others globally
  - Track your improvement over time

- 📊 **Game Review System**
  - Rewatch past games
  - Analyze your guesses and accuracy
  - Learn from mistakes and improve

- 🌐 **Educational Focus**
  - Improve real-world geography knowledge
  - Discover new countries, cities, and landmarks
  - Learn while playing

---

## 🧠 How It Works

Players are dropped into a random real-world location using street-level imagery. Your goal is to figure out where you are on the world map and place your guess as accurately as possible. The closer your guess, the higher your score.

You can play:
- Solo for practice
- Daily challenges for consistency
- Multiplayer for competitive matches

## 🖥️ Client

The frontend is built with:

- ⚛️ **React**
- 🎨 UI design assisted with AI tools for faster iteration and modern styling
- 📱 Responsive layout for desktop and mobile devices

The client handles:
- Map rendering
- User interactions
- Game UI and animations
- Real-time updates from the server

---

## ⚙️ Server

The backend is built with:

- 🚀 **Node.js + Express**
- 🔌 **Socket.io** for real-time multiplayer communication

The server handles:
- Matchmaking system
- Real-time multiplayer games
- Score calculation
- Map and game data management
- Room synchronization between players
